# Threat Analysis Labs

This folder focuses on analyzing potential threats using indicators, behavioral patterns, and threat intelligence concepts.

## Topics Covered
- Indicator of Compromise (IOC) analysis
- Threat actor behavior
- MITRE ATT&CK technique mapping
- Identifying false positives

## Skills Practiced
- Analytical thinking
- Pattern recognition
- Documentation of findings

## Goal
To strengthen analytical skills used in SOC and threat intelligence roles.
